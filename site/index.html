/ Features

Every detail, considered.

Pro-grade scheduling primitives, packed into a file you can email.

Critical Path Method

Forward + backward pass over your DAG. All four dependency types. Red-glow highlight on zero-float tasks.

Baseline drift

Snapshot the current schedule. Keep planning. The baseline lives under each bar so drift is visually obvious.

Cmd+K command palette

Every action — create, switch tools, change scale, swap themes, load templates, export — one keystroke away.

Five templates

Product Launch, Two-Week Sprint, Hiring Plan, Marketing Campaign, Blank. Realistic content, not lorem ipsum.

Vector export

PNG (1× or 2×), SVG, JSON, YAML, print-to-PDF. Crisp at any resolution. Designed for slides, not screenshots.

Offline-first

No network calls. No telemetry. No sign-in. Your data never leaves the file.

Self-contained file

Editor and data live in the same HTML. Reopen the file, keep editing. Share the file, share the work.

Drag-to-connect

Wire dependencies by dragging from one bar's edge to another. Choose FS, SS, FF, or SF.

Dark · Light · Print

Three first-class themes. Print mode is high-contrast, ready for paper or slide hand-outs.
